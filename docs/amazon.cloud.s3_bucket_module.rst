**********************
amazon.cloud.s3_bucket
**********************

**Create and manage S3 buckets**


Version added: 0.1.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Create and manage S3 buckets (list, create, update, describe, delete).



Requirements
------------
The below requirements are needed on the host that executes this module.

- boto3 >= 1.17.0
- botocore >= 1.20.0
- python >= 3.6
